# CutCut

Yet another elm project

### For start project

```
yarn install
node_modules/.bin/elm install
yarn start
```

For VS code users

_.vscode/settings.json_

```json
{
  "elm.compiler": "./node_modules/.bin/elm",
  "elm.makeCommand": "./node_modules/.bin/elm-make",
  "elm.formatCommand": "./node_modules/.bin/elm-format"
}
```
